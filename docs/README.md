# **Общая информация**

- - - 
Данный проект реализует геометрические функции площади и периметра/длины окружности для объектов типа круг и квадрат на языке Python.
Функции реализованы в отдельных файлах для каждом типа - circle.py для круга и square.py для квадрата.
- - -

# *Square.py*
## функция area(a)
* вычисляет площадь данного квадрата
#### Параметры
* float : a - длина стороны квадрата 
#### Возвращаемое значение
* float : площадь квадрата
#### Пример вызова
```
 area(2) -> 4
```
## функция perimeter(a)
* вычисляет периметр данного квадрата
#### Параметры
* float : a - длина стороны квадрата 
#### Возвращаемое значение
* float : периметр квадрата 
#### Пример вызова
```
 perimeter(2) -> 8
```

# Circle.py
## функция area(a)

#### Параметры

#### Возвращаемое значение

#### Пример вызова

## функция perimeter(a)
* вычисляет площадь данного квадрата
#### Параметры

 ```
perimeter(5) -> 31.41592653589793
  ```

# Rectangle.py
## функция area(a, b)
* вычисляет площадь прямоугольника
#### Параметры
* float : a - длина первой стороны прямоугольника 
* float : b - длина второй стороны прямоугольника
#### Возвращаемое значение
* float : площадь прямоугольника
#### Пример вызова
```
area(3, 4) // 12
```
## функция perimeter(a)
* вычисляет периметр прямоугольника
#### Параметры
* float : a - длина первой стороны прямоугольника 
* float : b - длина второй стороны прямоугольника
#### Возвращаемое значение
* float : периметр прямоугольника
#### Пример вызова
 ```
perimeter(3, 4) // 14
  ```
- - - 
# Triangle.py
## функция area(a, h)
* вычисляет площадь треугольника
#### Параметры
* float : a - длина основания треугольника
* float : h - высота треугольника, проведенная к основанию
#### Возвращаемое значение
* float : площадь треугольника
#### Пример вызова
```
area(6, 4) // 12.0
```
## функция perimeter(a, b, c)
* вычисляет периметр треугольника
#### Параметры
* float : a - длина первой стороны треугольника
* float : b - длина второй стороны треугольника
* float : c - длина третьей стороны треугольника
#### Возвращаемое значение
* float : периметр прямоугольника
#### Пример вызова
```
perimeter(3, 4, 5) // 12
```
